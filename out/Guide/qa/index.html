NTUCPC Guide Q & A

就是 Q&A。

作者
臺灣大學程式解題社

我可以用這份講義來學習「大學程式設計先修檢測（APCS）」嗎？

APCS 並不是資訊競賽，儘管他們的內容有所重疊，但其理念與多數資訊競賽並不相同。但由於 APCS 的「程式設計實作題」內容與資訊競賽相似，所以在資訊競賽表現優異的選手往往也能在 APCS 取得好成績。

只能說，若只是想要在 APCS 爭取好成績，參與資訊競賽並非是必要的，也存在著針對考試內容學習的方法；但反過來說，若對資訊競賽有不小的興趣，也可以考慮直接透過我們的講義開始學習，或許就能順手提高 APCS 的成績！

有沒有 Python 的教學？

很抱歉，沒有。

這是因為在資訊競賽中，主流的程式語言是 C++，大多數的競賽選手都習慣以 C++ 進行題目的撰寫，即使 Python 在少數狀況上能派上用場，我們也沒有辦法設計一份完整的 Python 教學來涵蓋所有內容。

若想要接觸資訊競賽的話，建議是一定要學習 C++。

我還有更多問題……

歡迎投稿至這個表單，我們會不定期檢查看看並回覆，若問題本身能夠解決許多人的疑惑的話，我們也會考慮將其放在這篇文章內。

當然，也歡迎直接加入臺灣大學程式解題社的官方 Discord 群在裡面發問！
